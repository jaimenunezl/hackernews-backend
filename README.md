# Hackernews Backend

Project Hackernews Backend for Reign Design

## Getting Started

### Prerequisites
- Node: 8.9.4

### Installing

* git clone https://github.com/jaimenunezl/hackernews-backend.git
* cd hackernews-backend
* npm install

### Run development

Run `npm run dev` for run in dev mode. Navigate to `http://localhost:3000/`.
